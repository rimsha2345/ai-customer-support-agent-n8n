# AI Customer Support Agent (n8n + Google Gemini)

This project is an AI-powered customer support chatbot built using n8n, Google Gemini, and Google Sheets. It handles real-time customer queries, inventory checks, FAQs, and order placement automatically.

## Workflow Overview
The workflow starts when a chat message is received. The AI Agent processes the message using the Gemini model, retrieves inventory and FAQs from Google Sheets, and saves customer orders automatically.

## Tools & Technologies
- n8n
- Google Gemini Chat Model
- Simple Memory
- Google Sheets
- AI Agent

## How It Works
1. User sends a chat message  
2. AI Agent processes the request  
3. Gemini generates a natural language response  
4. Inventory and FAQs are fetched from Google Sheets  
5. Orders are saved automatically to Google Sheets  
6. User receives an instant reply
